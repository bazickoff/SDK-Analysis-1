# SDK-Analysis
这个工具主要是为了分析 apk 中所集成的 SDK 用的。
## 使用方法
1.将要查看的apk 放在工程里的 apk 目录下
2.将用于拆包的 baksmali.jar 放在工程的根目录下
3.指定输出log文件的地址，如‘D:\\My\\Python\\SdkAnalysis\\filelistlog.txt’
4.启动项目
